<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
In this lab, we are going to walk through how to create an Active Directory home lab environment using Oracle Virtual Box.  Configuring and running this lab is meant to help me develop an understanding of how active directory and windows networking works.  After a few follow throughs I was able to build this from memory without the assistance of a tutorial video.  As a systems administrator, this was useful in rounding out all the practical on the job experience I have maintaining such environments.
